About

DIYA pool is a stake pool running on cardano blockchain network.
The pool is setup by Suracheth Chawla on February 21 of 2021, to promote the decentralization of cardano network.
The id and ticker of the pool are bf8892369c5922d9d5e404ab733c9dbf75b6fb842fa1c09f46c60616, DIYA respectively.
Suracheth Chawla is also the owner of the Cardano in Thai Language youtube channel, in which he occasionally makes video about cardano in Thai language and post them there.

DIYA pool is proud to be a member of Single Pool Alliance, where we promote stake pools' provider to run only one single pool,
and First 2 Lifetime Block community, so to make small stake pool providers not run in loss.
